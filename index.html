Overview

The iRECYCLE app facilitates recycling of textile waste in Kenya, targeting second-hand clothes traders within Nairobi and its environs. The goal of this application is to contribute to environmental sustainability, especially by reducing the waste that ends up at the Dandora dumpsite from unsold second-hand clothes.

Why Trashion Management Matters

Textile waste in Nairobi's markets piles up fast, contributing heavily to dumpsites like Dandora. Through iRECYCLE, traders and agents turn unsold clothes into valuable recycled materials, protecting our local environment.

User Guide

For Traders: Getting Started

  1. Download the app: The app will be available for download to select traders for testing via a provided link.
  2. Create an account: Open the app and click "Sign Up." Enter your name, phone number, location, and password. Click 'Sign Up' to create your account. If you already have an account, log in with your phone number and password.
  3. Request a pickup: On the home page, tap on "New Pickup Request." Fill in the type of textile, estimated weight, and upload a photo. Tap "Submit." You will receive a notification once pickup is scheduled.
  4. View price catalogue: Access the price catalogue screen to see types of materials and current price per kilogram.
  5. View pickup history: Track past and pending pickups, including date, weight, status, and type of textile.
  6. Track payment history: View date, amount, and status of your payments.
  7. Manage your profile: Update your name, phone number, and current market as needed.

Agent Guide

  1. Download the app: Download link provided.
  2. Create an account and log in: Enter your details and sign up. Log in using your phone number and password.
  3. View collections: On the home page, see pending pickups for your assigned markets; view pickup details and map location for navigation.
  4. Price catalogue: Reference buying rates for different materials.
  5. Request payment: Request payment after a pickup by entering the trader's phone number and sending a payment request.
  6. Pickup history: Track all collections in your assigned market with date, type, weight, and trader's name.

Developer Guide

System Requirements

  • Hardware: 8GB RAM minimum
  • Software: Android Studio 2022.3+, npm, Python 3.8+, Django 4.2+, Django REST Framework 3.14+, PostgreSQL 13+, Postman
  • Dependencies:
    • Python: pip install django djangorestframework
    • JavaScript: npm install

Project Setup

  1. Android App: Clone repo, open in Android Studio, configure emulator, sync Gradle, and build project.
  2. React Dashboard: Clone repo, install dependencies, and run development server on http://localhost:3000.
  3. Django Backend: Clone repo, create & activate virtualenv, install requirements, apply migrations, and run server on http://localhost:8000.

API Integration

  • Manage traders: /api/traders/
  • Manage agents: /api/agents/
  • Manage companies: /api/companies/
  • Unified user creation: /api/user-union/
  • Payments: /api/payments/
  • Initiate M-Pesa payment: /api/stk-push/
  • Daraja API: /api/daraja/
  • Fabric catalogue: /api/catalogue/
  • Collections: /api/collections/
  • Clothes listings: /api/clothes-listings/
  • Market locations: /api/markets/

M-Pesa Integration (Daraja)

Payload (STK Push):
{
  "phone_number": "254712345678",
  "amount": "1000.00",
  "order_item": ["Textile Waste"],
  "account_reference": "TX12345",
  "transaction_desc": "Payment for pickup"
}

Use OAuth 2.0 credentials from Safaricom for authentication.

Testing

  • Test API endpoints using Postman (GET /api/traders/, POST /api/stk-push/)
  • Write backend tests for Django using PyTest

Frequently Asked Questions (FAQ)

Who can use iRECYCLE?
Second-hand clothes traders in Nairobi and agents working with recycling companies.
How do I get paid for my textile waste?
Payments are processed via M-Pesa after successful pickup and verification.
How do I update my trader profile?
Go to "Manage profile" within the app and edit your details.
